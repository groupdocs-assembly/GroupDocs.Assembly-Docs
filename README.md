# GroupDocs.Assembly - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Assembly](https://docs.groupdocs.com/assembly/) static documentation.

## About GroupDocs.Assembly

[GroupDocs.Assembly](https://products.groupdocs.com/assembly/) is a document automation SDK that enables you to generate documents by assembling templates with data from various sources.

Available for .NET, Java, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/assembly/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
